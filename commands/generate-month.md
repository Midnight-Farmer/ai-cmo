# Generate Monthly Plan

The user wants a monthly content strategy. Read `skills/ai-cmo/references/planning.md` § Monthly Plan for the full procedure.

**Arguments:** The user may specify a client name.

**Process:**
1. Identify the client (ask if ambiguous)
2. Read the client's `.claude/CLAUDE.md` and all knowledge files
3. Review performance data and what's working
4. Generate month-level strategy with 4-week breakdown
5. Include hypotheses to test, production planning, metrics targets
6. Save to `marketing/outputs/monthly-briefs/`
